# foliosAtendidosSD
Reporte de folios atendidos de Service Desk
Requerimientos
Hacer la conexion a la base de datos de service desk, desde nuestro glassfish con el nombre de jdbc/reporteFoliosSD

Para este proyecto Spring se debe clonar el repositorio o bien descargarlo a su maquina local
Una vez descargado se debera importar al workspace de nuestro netbeans y ejecutarlo (No se ocupa nada m√°s)
Ya que se ejecuta se mostrara en el navegador un input de tipo date en el cual le agregaremos la fecha del jueves anterior o de la fecha a la cual se requiera el reporte

Ya que se agrego la fecha se le da clic en "Generar" y nos descargara el reporte en xlsx
